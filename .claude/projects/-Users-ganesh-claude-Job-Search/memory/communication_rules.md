**Why:** User wants every output to sound human-written and specific to each role/company. Generic AI-sounding language is a dealbreaker.

**How to apply:** Every cover letter, outreach message, email, and evaluation narrative must follow these rules. If a sentence sounds like an LLM wrote it, rewrite it.
